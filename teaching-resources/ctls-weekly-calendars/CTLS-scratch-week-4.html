Programming with Scratch — Week 4

Unit: Intermediate Scratch

Welcome back from Spring Break! This week you start a brand new project — a falling-objects game — while reviewing the skills you learned in Weeks 1–3. The new skill for the week is cloning, which lets one sprite create many copies of itself.

Day Date Topic Summary
16 Mon 4/13 Welcome Back — New Game Students review motion, loops, conditionals, and variables by building a playable falling-objects catch game from scratch, adding a player sprite, a falling sprite, and a score variable.
17 Tue 4/14 Variables Deep Dive Students add a speed variable that increases the fall rate with each catch and a lives variable that ends the game after three missed objects.
18 Wed 4/15 Clones Students replace the single falling object with a clone factory that spawns multiple independent falling objects, then add a second danger sprite whose clones the player must dodge.
19 Thu 4/16 Game States Students add a clickable start screen and a game-over screen using broadcasts, coordinating all sprites so the game moves cleanly between the start, playing, and game-over states.
20 Fri 4/17 Quiz & Share Your Game Students review, then take the Intermediate Scratch unit quiz in CTLS; afterward they make final polish tweaks and share their finished game to the class Scratch studio.
Graded Assignments
  • Falling-Objects Game (due Friday 4/17) — A complete Scratch game with player movement, clones, scoring, lives, game states, and at least one sound effect. Share your project on Scratch and add it to the class studio before class.
  • Intermediate Scratch Unit Quiz (Friday 4/17, in class) — Quiz in CTLS covering vocabulary, booleans, loops, conditionals, variables, broadcasts, and clones. Review the study guide and practice questions linked from Day 18.
