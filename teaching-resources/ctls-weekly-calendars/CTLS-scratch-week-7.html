Programming with Scratch — Week 7

Unit: Python and the Terminal

This week you move from Scratch to Python. Monday and Tuesday cover the basics — installing Python, using VS Code, and writing your first interactive program. Wednesday through Friday we use Python to explore real-world data with the pandas and matplotlib libraries: starting with a Pokémon dataset, then opening it up so you can pick a dataset that interests you and answer your own questions about it.
