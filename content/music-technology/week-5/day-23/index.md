## Work Session: Ode to Joy in Musescore

Open Musescore and create a new score. Follow the steps below to transcribe the Ode to Joy melody.

### Set Up Your Score

1. Open **Musescore** and choose **New Score**.
2. Select a **Piano** out of the **Keyboard** family of instruments.
3. Set the time signature to **4/4** and the key signature to **G major** (one sharp: F#).
4. Title the score: **Ode to Joy — [Your Name]**.

### Enter the Notes

Press **N** to enter note input mode. Use the number keys to set durations:

| Key | Duration |
| --- | -------- |
| 5   | Quarter note |
| 6   | Half note |
| 7   | Whole note |

Type the letter name of each note to enter it or use the mouse to click on the staff.

**The melody starts on B4.**

Work measure by measure. Use the sheet music below. If you lose your place, click the last correct note and continue from there.

{{< button text="View Sheet Music" >}}ode-to-joy.svg{{< /button >}}

### Check Your Work

When you finish entering all the notes:

1. Press **Escape** to exit note input mode.
2. Press **Ctrl+Home** to go back to the beginning.
3. Press the **spacebar** to play back the score.
4. Does it sound like Ode to Joy? If something is off, find the wrong note and fix it.
