## Computer Programming with Scratch

aligned to Georgia's Foundations of Computer Programming standards (MS-CS-FCP).

Introduces fundamentals of computer programming and computational thinking to 6th–8th graders using MIT's block-based Scratch language, with no prior programming experience required.

45 days across 9 weeks

## Units so far...

### Unit 1: Introduction to Scratch (Week 1, Days 1–5)
Computer lab setup, Scratch interface orientation, sequencing, events (green flag), motion blocks, sprite art tools, and maze design/implementation.

### Unit 2: Conditionals & Control Flow (Week 2, Days 6–10)
Keyboard input events, `if` conditionals, collision detection (touching color/sprite), flow diagrams, `forever` and `repeat` loops, and code efficiency.

### Unit 3: Boolean Operators & Gravity/Platformer (Week 3, Days 11–15)
Boolean operators (`and`, `or`, `not`), velocity-based movement, gravity simulation, platform and wall collision (step-up pattern), variables for game state, platformer game design, and a Day 15 introduction to CLI via Terminal and Minecraft Education Edition. A multiplayer server activity is referenced as a post-spring-break follow-up.

### Concepts Covered
- **Core:** Sequences, events, loops, conditionals, Boolean operators, variables, data types.
- **Scratch-specific:** Sprites, costumes, stage/backdrop, block categories, debugging, algorithms.
- **Advanced for middle school:** Velocity-based movement, gravity simulation, multi-type collision detection (step-up pattern for walls), game state management, scoring/respawning.

### Projects
1. **Maze Game** — Students design on paper, build art in Scratch, then add keyboard controls and wall collision in Week 2.
2. **Platformer Game** — Multi-day progression: gravity (Day 12) → platforms with collision (Day 13) → collectibles and scoring (Day 14) → complete level with hazards.
3. **Box Art Project** — Offline activity designing professional front and back game covers (title, artwork, studio logo, age rating, tagline, description, feature list).
